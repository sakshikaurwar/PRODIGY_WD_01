# PRODIGY_WD_01

## Interactive Navigation Menu

This project was developed as part of Task 01 of the Web Development Internship at Prodigy InfoTech.

### Description

An interactive navigation menu built using HTML, CSS, and JavaScript. The navigation bar remains fixed at the top of the page and changes its appearance when the user scrolls or hovers over menu items, providing an engaging and responsive user experience.

### Features

* Fixed navigation bar
* Hover effects on menu items
* Dynamic style changes on scroll
* Smooth transitions and animations
* Responsive design
* User-friendly interface

### Technologies Used

* HTML
* CSS
* JavaScript

### Learning Outcomes

* Creating fixed navigation menus
* Implementing hover effects using CSS
* Handling scroll events with JavaScript
* DOM manipulation
* Building responsive and interactive web interfaces
